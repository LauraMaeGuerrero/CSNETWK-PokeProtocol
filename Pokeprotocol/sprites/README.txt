SPRITES DIRECTORY

    For sprites to work, you need GIF format images (Tkinter native support).

    Required player sprites (GIF format):
    - player1.gif
    - player2.gif  
    - spectator.gif

    Required Pokémon sprites (GIF format, 50 total from Bulbasaur to Diglett):
    - bulbasaur.gif
    - ivysaur.gif
    - venusaur.gif
    - charmander.gif
    - ... etc.

    Tkinter only supports GIF natively. Convert PNG/JPG to GIF using online tools.
